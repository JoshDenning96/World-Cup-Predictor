About the model

How the simulation works and what the numbers mean.

Monte Carlo simulation. Each run simulates the full FIFA 2026 tournament a user-selected number of times from the current state. Probabilities (e.g. "32% chance of winning") are simply how often each outcome occurred across all simulations — not a single prediction. More simulations give smoother, more stable probabilities.
Elo-based match model. Each match is resolved using FIFA Elo ratings. The rating difference between two teams determines the win/draw/loss probabilities. Draws are possible in the group stage; in knockout rounds a drawn match is resolved as a 50/50 coin flip (simulating a penalty shootout).
Host-nation advantage. USA, Canada, and Mexico each receive a +50 Elo point boost to reflect home-crowd advantage. This is set at half the typical neutral-to-home shift (~100 points) because the tournament is spread across 16 venues in three countries rather than a single host nation. The boost applies in both the group stage and knockout rounds.
CONMEBOL adjustment. A −45 Elo point offset is applied by default to all qualified CONMEBOL teams. This acts as a scepticism dial, reducing South American teams' projected strength relative to their raw Elo rating.
Actuals + simulation mode. Once real matches have been played, enter the results in the actuals panel. The simulation then fixes those outcomes and runs forward from the current bracket state, so probabilities update as the tournament progresses.
Interpreting the bracket. The bracket shows the most likely path for each team based on simulation probabilities. Box border and left-bar colour shift from grey (low win probability) to gold (high). A green dot indicates the position was confirmed by an actual result, not a projection.

How to use

Export the latest simulation data and host this dashboard locally.

1. Run the export script: python3 scripts/export_simulation_json.py
2. Start the interactive server from the repo root: python3 server.py
3. Open http://127.0.0.1:8000 in your browser.
