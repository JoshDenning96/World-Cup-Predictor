How to use

Export the latest simulation data and host this dashboard locally.

1. Run the export script: python3 scripts/export_simulation_json.py
2. Start a local server from the website folder: python3 -m http.server 8000 --directory website
3. Open http://127.0.0.1:8000 in your browser.
